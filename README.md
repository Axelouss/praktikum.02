# praktikum.02
Praktikum ini mempelajari cara menggabungkan HTML dan CSS untuk membuat halaman web sederhana. HTML digunakan untuk membuat struktur dan isi halaman, sedangkan CSS digunakan untuk mempercantik tampilan seperti warna, posisi, dan ukuran teks. Hasilnya, halaman web menjadi lebih menarik dan rapi.
